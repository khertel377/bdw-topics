Clearfix

There are three ways to do a clearfix, or clearing floats after a float-collapse has occurred.

  1. Creative a div and style it inline. Do inline for a blog, if it's a part of a post and you're floating a couple images and then clearing the float at the end. "Class=clear" means nothing in google reader or bloglines, etc. Inline will still work in a feedreader. This solution is frowned upon by semantic purists because an empty div has no adds no contextual meaning to the content.
  2. Create a div class and style that class in CSS. Cleans up your code a little bit (not by much)
  3. Style the wrapper itself with overflow: auto; or overflow; hidden
  4. "The Easy Clearing Method" uses a css pseudo selector (:after) to clear floats, by applying a class like "clearfix" to the parent element and then adding the CSS .clearfix:after { content: "."; visibility: hidden; display: block; height: 0; clear: both; }
